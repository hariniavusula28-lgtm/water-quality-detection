# Water Quality Monitoring and Forecasting System

## 📌 Project Overview
The **Water Quality Monitoring and Forecasting System** is an intelligent web-based project developed to analyze water quality parameters and predict whether water is **safe or unsafe for drinking purposes**.  

This system uses **Machine Learning algorithms** to classify water quality based on different chemical and physical properties of water. It helps users, industries, and authorities monitor water quality efficiently and make faster decisions.

---

## 🎯 Objectives
- To monitor water quality using input parameters.
- To predict whether water is safe for drinking or not.
- To compare machine learning algorithms for better accuracy.
- To reduce manual testing time and effort.
- To provide an easy-to-use web interface for users.

- pH Value
- Hardness
- Solids
- Chloramines
- Sulfate
- Conductivity
- Organic Carbon
- Trihalomethanes
- Turbidity

- Upload Dataset / Enter Water Parameters
- Water Quality Prediction
- Safe / Unsafe Result Display
- Accuracy Comparison of Algorithms

- Admin Dashboard

---

## 🖥️ System Workflow

1. User logs into the system.
2. User enters water quality parameters.
3. Data is preprocessed.
4. Machine learning model analyzes the data.
5. System predicts:
   - **Safe Water**
   - **Unsafe Water**
6. Result is displayed to the user.
